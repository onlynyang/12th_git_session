# 깃 세션 내용 요약정리

Github 사용법
1. Github에서 레포지토리 생성
2. 폴더 생성하고 등록하기
3. 레포지토리 주소를 복사해서 연결하기
4. 파일을 수정하고, 편집하기
5. 수정 내용을 반영하고 push로 저장소에 넣기 
    
Django 사용법
1. Github에서 레포지토리 생성(리드미 생성X)
2. 폴더 생성하고 등록하기
3. Git에 연동하기
4. 가상환경 설치(pip3 install pipenv)
5. 가상환경 켜기(pipenv shell)
6. django 설치(pip install django)
4. 초기 설정, 앱 생성, 서버 실행 순서로 진행하기
